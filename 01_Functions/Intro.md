# Functions in Python

**Function** is the block of code that can reuse in the program several times. It takes data values as arguments and returns a value.

---

In this section we have examples of:

-   Positional and Keyword Arguments.
-   Return value.
-   Local and Global Scopes.
-   Doc-string.
-   Lambda Functions.
-   Hier-Order Functions.
-   map(), filter(), reduce() functions.
-   Decorators.
-   Generators.

---

We have examples of:

-   Sum of Prime numbers.
-   Fibbonacci Series Generators.
