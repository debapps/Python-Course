# Modules, Packages and Library

In Python, the words - _modules, packages and libraries_ are used in simillar ways. But there are slight difference in those words.

#### Module

Module bundles the functions, variables and classes into a single python file (with .py extension). The functions, variables, and classes often share the simillar and related functionality of the application.

#### Package

A package is the folder containing several modules or sub-packages those share coherent functionalities of an application.

A package often initialized with a `__init__.py` file.

#### Library

A Library is the folder that contains a group of packages that solve a particular aspect in the application. A library is also initialized with a `__init__.py` file.
